# Text

Displays the string representation (`toString()`) of the first argument:

```kotlin
text("Hello World")
text(12)
```

### Instructions

```kotlin
text("a", fontName("Courier New"))
text("a", fontSize(12))
text("a", lightFont)
text("a", letterSpacing(- 2))
text("a", noTextWrap)
```
